1. Multiple Linear Regression Model

(1) A First Look

  • : 설명변수(Explanatory Variable, Predictor Variable, Regressor)

  • : 편회귀계수(Partial Regression Coefficients)

2 개 이상의 설명변수를 포함하며, 각 설명변수가 선형결합된 형태의 모형을 다중선형회귀모형이라 한다.

이때 는 모형의 파라미터(Parameter)로서, 절편 와 개의 편회귀계수(Partial Regression Coefficients) 로 이루어져 있다.

Note

편회귀계수의 개수를 개라 하면, 파라미터의 개수는 개이다. (까지 포함)

편회귀계수 는 다른 설명변수 가 모두 고정인 상태에서 만 변할 때 반응변수 가 변하는 정도를 나타낸다.

이때 한 설명변수가 다른 설명변수의 함수여도 상관이 없다.

은 0 이상 1 이하의 값을 가지는데, 1 에 가까울수록 회귀모델이 모든 변동성을 완벽하게 설명하고, 0에 가까울수록 모델이 아무런 변동성도 설명하지 못한다.

Hidden Extrapolation

주어진 에서 평균반응 또는 새로운 반응 를 추 정할 때, 오른쪽 그림과 같이 가 각 성분 의 범위 안에는 포함되어 있지만, 이들의 결합확률분포 밖에 있는 경우가 있을 수 있다. 이러한 경우에의 추정을 Hidden Extrapolation이라 한다.

데이터를 구성하는 임의의 에 대해 를 만족하는 최소의 가 존재하면, 주어진 에 대해 구한 를 가지고 외삽 여부를 결정한다. 이면 타원영역 바깥에 있어 외삽을 해야 한다.
